# Online Cinema Booking Platform

Microservices-based cinema booking platform built using Spring Boot and React.

## Architecture Overview

- Spring Boot Microservices
- React (Bookers UI & Backoffice UI)
- Spring Cloud Gateway
- Eureka Service Discovery
- Keycloak Authentication
- RabbitMQ Messaging
- Stripe Payment Integration
- MinIO Object Storage
- PostgreSQL (Schema per service)

---

## MVP Scope

This project implements a minimal viable version of the cinema booking system demonstrating:

### Infrastructure
- Service Discovery using Eureka
- API Gateway routing and JWT validation
- Keycloak authentication

### Core Features
- Branch management
- Screen and show scheduling
- Seat selection and holding
- Real-time seat updates via WebSocket
- Stripe payment checkout
- Booking confirmation via webhook
- Ticket generation and validation
